# numbrs

Time-series metrics and dashboards for the Nostr ecosystem.

## Stack

- Vite + React + TypeScript
- shadcn/ui + Tailwind CSS
- Supabase (auth, database, edge functions)

## Local development

```sh
git clone https://github.com/satsdisco/numbrs
cd numbrs
npm i
npm run dev
```

## Deploy

Hosted on Vercel at [numbrs.lol](https://numbrs.lol).
